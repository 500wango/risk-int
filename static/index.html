中国企业海外投资案例库

中石油中亚天然气管道项目

企业及项目概况

中国石油天然气集团是中国最大的油气生产商和供应商。

中亚天然气管道是中国第一条跨国天然气管道，总投资约100亿美元，全长1833公里，途经土库曼斯坦、乌兹别克斯坦、哈萨克斯坦到达中国新疆，年设计输气能力550亿立方米。

出海模式

资源合作+基础设施投资

采用"资源换贷款"模式，中国提供贷款和管道建设，换取长期天然气供应合同。

核心举措

1. 多国协调机制：建立四国政府间协调委员会
2. 分段建设管理：按国别分段建设，降低政治风险
3. 技术标准统一：推广中国管道建设标准和技术
4. 本地化采购：在沿线国家采购超过50%的物资和服务

金融服务支持

• 政府间贷款：中国提供250亿美元贷款，换取天然气供应
• 出口信贷：中国出口信用保险公司提供政治和商业风险保险
• 项目融资：组建国际银团提供项目融资
• 人民币结算：部分交易采用人民币结算，降低汇率风险

风险应对

1. 地缘政治风险：保持中立，与各国政府均建立良好关系
2. 安全风险：与沿线国家安保部队合作，建立分段安保责任制
3. 价格波动风险：采用长期合同与现货价格挂钩的定价机制
4. 技术标准差异：推动中国标准与当地标准的互认

存在的问题与不足

• 沿线国家政策稳定性差异大
• 极端气候和复杂地质条件增加建设难度
• 多国协调效率有待提高
• 人民币国际化程度不足影响结算便利性

成果与意义

项目自2009年投产以来，已累计向中国输送天然气超过3000亿立方米，占中国进口天然气总量的30%以上。

管道建设带动了中国装备、技术和标准"走出去"，创造了数万个当地就业岗位。

该项目建立了中国与中亚能源合作的新模式，保障了国家能源安全，是"一带一路"能源合作的典范。
